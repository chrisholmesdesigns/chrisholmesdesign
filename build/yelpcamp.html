What Is YelpCamp?

This full-scale web application allows users to add new campgrounds to a database. Users can also comment on existing campgrounds and delete their previous posts. In order to post, users must sign up and log in. The user interface is intuitive and straight forward with limited frills for a stress-free user experience.

Target Audience

  • Male & Female
  • Ages 22-36
  • Focused on Health and Wellness

Influential Idea

Rate and Review Campsites

Role

Research, Wireframing, Front & Back End Development, Design Execution

Category

Dynamic Web Application

Research

UX Research

Yelp is the inspiration for the web application. The web app utilizes features such as leaving a comment or signing up for an account. This research helped me to understand what users expected and what was needed from the UI.

Challenge

Design an app that connects users in a more involved way with their local community. There needed to be a reward system so that users would feel emotionally and financially attached to the app in order to create trust. Wrangle is a way of allowing customers to have a mutually beneficial relationship with their neighborhood.

Solution

Created an app that intuitively led the user through the process of signing up and receiving rewards. Made the user flow straight forward and allowed for each screen to be self-explanitory in nature. In this way, the focus is on the experience the customer is having at the local business and not on figuring out how to use the app.

The Process

Back End Development

The back end is coded using the MEN stack (no angular in this one). MongoDB is used for the database to enhance the dynamic capabilities of the site while express and node are used to power the routing.

Front End Development

The front end utilizes ejs to make sure it properly can connect information from the database to the site. This ensures that the user has a seamless experience when they revisit the site or refresh the page.
