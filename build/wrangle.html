What Is Wrangle?

WRANGLE is an app for locating businesses in the Normal Heights neighborhood of San Diego. It starts by giving the user an optional tutorial on how the app works and follows up with a log in/sign up screen. From there, the user can see local businesses participating in the app and can check in to a location by scanning a QR code provided to them. The user is awarded points for supporting local businesses and these points can then be redeemed for rewards. There are also coupons that are free to users of the app.

Challenge

Design an app that connects users in a more involved way with their local community. There needed to be a reward system so that users would feel emotionally and financially attached to the app in order to create trust. Wrangle is a way of allowing customers to have a mutually beneficial relationship with their neighborhood.

Solution

Created an app that intuitively led the user through the process of signing up and receiving rewards. Made the user flow straight forward and allowed for each screen to be self-explanitory in nature. In this way, the focus is on the experience the customer is having at the local business and not on figuring out how to use the app.
